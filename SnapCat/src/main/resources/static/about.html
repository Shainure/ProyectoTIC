SnapCat



“SnapCat”, un sitio web inspirado en la aplicación móvil “Snapchat”. A diferencia de su versión original, este sitio está dedicado para subir fotos / selfies de tu mascota; específicamente gatos. ¡Porque ellos también merecen su espacio!

El objetivo es simple, en SnapCat puedes registrarte y subir fotos para así mantener presente tu galería personal o simplemente publicar fotos de forma anónima, cuando creas nuevo contenido podrás poner información tal como el nombre de tu mascota, su género, edad y compartir algo más!
